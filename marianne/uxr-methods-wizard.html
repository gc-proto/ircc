If you want to assess findability of the live site:

  • Tree test: Test the labels in several levels of the current information architecture (IA). Find out if people go down the right path to find something.
  • First-click test: Test the labels, accompanied by the visuals and surrounding information, in the current IA. You're only testing the user's initial click, but you can test a different page per task.
